当店「六古窯」について

２００１年５月、
名古屋市瑞穂区に
オープンした喫茶店です。

落ち着いた雰囲気、
心地よいBGM、
そして美味しい珈琲。
お客様ひとりひとり、
思い思いの時間を
お過ごしください。
